Setup instructions

        
Node 18+

      

      

        

          

            
Exact install command

            
npm install

          

          

            
Run server

            
node server.js

          

          

            
Open frontend

            
Open public/index.html in a browser (file:// is OK).

          

          

            
Required env vars

            

              METALS_API_KEY

              EIA_API_KEY

              PROVIDER_REFRESH_MS
            

          

          

            
Example (macOS/Linux)

            

              export METALS_API_KEY="YOUR_KEY"

              export EIA_API_KEY="YOUR_KEY"

              export PROVIDER_REFRESH_MS="5000"

              node server.js
            

          

          

            
Example (Windows PowerShell)

            

              $env:METALS_API_KEY="YOUR_KEY"

              $env:EIA_API_KEY="YOUR_KEY"

              $env:PROVIDER_REFRESH_MS="5000"

              node server.js
            

          

          

            
Backend endpoint

            
http://127.0.0.1:3000/prices

          

        


        

          If the UI shows offline, verify the server is running and you can open
          http://127.0.0.1:3000/prices directly in the browser.
        


        

          
          
          
          
        


        
No API keys are present in frontend; keys are read from environment variables on the backend only.
